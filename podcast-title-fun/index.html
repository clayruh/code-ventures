Welcome to the Coding Challenge!

This uses the browser's built in developer tools and console to interact with the code.

Let's get started!

  1. Open your browser's console. In Google Chrome, select the menu View -> Developer -> Javascript Console
  2. With the developer tools open, reload the browser page and you should see 
    Welcome! Let's fetch some podcast data!
  3. Take a look at the source code for this page "index.html" and the two javascript files (supporting_functions.js and challenge_functions.js)
  4. Fill in the functions for the given challanges. When you're done let us know!
